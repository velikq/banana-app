# Сборка приложения для Windows

## Один раз перед сборкой

Нужна Windows и Node.js LTS (20+). Откройте PowerShell в папке проекта и установите зависимости:

```powershell
npm install
```

## Создание установщика

```powershell
npm run dist
```

Готовый NSIS-установщик появится в папке `dist`. Сборка использует Electron из `node_modules/electron/dist`, поэтому не скачивает его повторно. Если `npm install` был прерван, сначала повторите его, чтобы эта папка восстановилась.

Установщик ставит приложение по умолчанию в профиль текущего пользователя (обычно `AppData\Local\Programs\banana.pots pro-vip`). Это важно: приложение сохраняет данные рядом со своим `.exe` и должно иметь право записи в эту папку.

Во время установки можно выбрать другую папку. Не выбирайте `C:\Program Files` без запуска приложения с правами администратора — Windows обычно запрещает приложению записывать туда настройки и изображения.

## Резервная сборка без установщика

Если сеть не даёт `electron-builder` скачать компоненты NSIS, соберите готовую папку приложения:

```powershell
npm run dist:unpacked
```

Готовая программа будет в `dist\\win-unpacked`; запускайте `.exe` из этой папки. Её можно целиком скопировать в любую доступную для записи папку. Папка `data` будет создана рядом с `.exe`. Эта команда не использует NSIS и подходит как обходной путь при ошибках `ECONNRESET` и TLS-обрывах во время `npm run dist`.

Для ZIP-сборки проект использует установленный системный 7-Zip по пути `C:\\Program Files\\7-Zip\\7z.exe`, поэтому скачивать 7zip-компонент сборщику не требуется.

## Данные после установки

В папке с установленным `.exe` приложение создаёт отдельную папку `data` и хранит в ней:

- `input` — «Входящие» и референсы;
- `output` — результаты генерации;
- `projects` — превью и список проектов;
- `config.json` — настройки интерфейса, модели и окна;
- `.env` — API-ключи;
- `kie-upload-cache.sqlite` — кэш загруженных в Kie изображений.

При обновлении приложения не удаляйте папку `data`, если хотите сохранить проекты, изображения, настройки и ключи. Перед переустановкой в другую папку скопируйте её рядом с новым `.exe`.
